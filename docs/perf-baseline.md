# perf-baseline

## How to measure (manual, Windows Task Manager is OK)
Record after:
- 10 minutes idle (panel hidden)
- 5 open/close cycles of panel
- 20 CRUD operations test (optional)

## Metrics
- Memory (Working set)
- CPU (idle average)
- Panel show latency (subjective + stopwatch if possible)
- Sync time (first sync, subsequent sync)
